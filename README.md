## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Tech Stack

- Next.js 13+ (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- React Icons
- Leaflet (for future map integration)
